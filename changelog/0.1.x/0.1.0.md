# 0.1.0 — 2026-05-25

## Added

- **`epa_search_facilities`** — search EPA-regulated facilities by location, program, and compliance status via ECHO `get_facility_info`; returns `RegistryID`, county FIPS, coordinates, per-program flags, and TRI totals
- **`epa_get_facility`** — full compliance profile aggregated from 3–5 parallel ECHO DFR endpoints; uses `Promise.allSettled` for partial-success resilience
- **`epa_search_violations`** — area-level civil and criminal enforcement case search via ECHO `case_rest_services`
- **`epa_get_air_quality`** — current and forecast AQI by ZIP or lat/lng via AirNow; ~1 hour response cache
- **`epa_get_tri_releases`** — per-chemical release breakdown by medium for a single TRI facility via DMAP `tri.tri_reporting_form`
- **`epa_search_tri_releases`** — area TRI emitter search by state/county and year
- **`epa_search_superfund`** — Superfund site search by location or lat/lng + radius via DMAP `sems.envirofacts_site`
- **`epa_search_water_systems`** — drinking water system search by state/ZIP via DMAP `sdwis.water_system`
- **`epa://facility/{registry_id}`** resource — compliance profile by EPA Registry ID
- **`epa://superfund/{site_id}`** resource — Superfund site record by SEMS site ID
- `EchoService`, `DmapService`, `AirNowService` — service layers for ECHO, Envirofacts DMAP, and AirNow APIs
- `AIRNOW_API_KEY` env var (required for air quality tool); ECHO and DMAP are unauthenticated
